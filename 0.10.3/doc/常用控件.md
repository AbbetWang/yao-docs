# 常用控件配置

## 图片显示与编辑

在相关的`model`中设置字段类型为`json`，

```json
{
  "label": "照片",
  "name": "photo",
  "type": "json",
  "nullable": true
}
```

在`table/form`定义中设置图片相关的控件属性。

注意:在 form 里，需要同时设置`view/edit`属性

```json
{
  "图片": {
    "bind": "images",
    "edit": {
      "type": "Upload",
      "compute": "Upload",
      "props": {
        "filetype": "image",
        "$api": {
          "process": "fs.system.Upload"
        }
      }
    },
    "view": {
      "type": "Image",
      "compute": "Download",
      "props": {
        "style": {
          "width": 36,
          "height": 36,
          "borderRadius": "50%"
        }
      }
    }
  }
}
```

## 设置控件的必输，使用编辑控件的配置属性`itemProps`

```json
{
  "名称": {
    "bind": "name",
    "edit": {
      "props": {
        "itemProps": {
          "rules": [{ "required": true }]
        },
        "placeholder": "请输入 名称"
      },
      "type": "Input"
    },
    "view": {
      "type": "Text"
    }
  }
}
```
